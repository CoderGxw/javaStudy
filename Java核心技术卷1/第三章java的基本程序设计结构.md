# 数据类型

## 四种整型

| 类型  | 存储需求 | 取值范围                                  |
| ----- | -------- | ----------------------------------------- |
| int   | 4字节    | -2147483648～2147483647（约20亿）         |
| short | 2字节    | -32768～32767                             |
| long  | 8字节    | -9223372036851775808～9223372036854775807 |
| byte  | 1字节    | -128～127                                 |

在java中，整型的范围与机器无关

## 两种浮点类型

| 类型   | 存储需求 | 取值范围        |
| ------ | -------- | --------------- |
| float  | 4字节    | 有效位数为6-7位 |
| double | 8字节    | 有效位数为15位  |

注意：浮点数值不适合用于金融计算中，金融计算中应该使用BigDecimal

## char类型

char类型表示单个字符。

## boolean类型

boolean类型只有两个值 true和false，用来判定逻辑条件
